## 第一阶段：STM32 裸机开发（当前 🎉）

### 网课推荐

| 优先级 | 课程 | 链接 |
|:--:|------|------|
| ⭐⭐⭐ | **江协科技 STM32 HAL库重制** | [B站搜 BV1zRhEzjE6F](https://www.bilibili.com/video/BV1zRhEzjE6F/) |
| ⭐⭐⭐ | **正点原子 手把手STM32 HAL库** | [B站搜 BV1eC8wzdEJQ](https://www.bilibili.com/video/BV1eC8wzdEJQ/) |
| ⭐⭐ | 野火 7天入门STM32 | [B站](https://www.bilibili.com/) |

> **建议**：江协科技为主线（短平快），正点原子作补充（细节深）

### 要学到什么地步

| 外设 | 熟练度 | 标准 |
|------|:--:|------|
| GPIO | 🔴 精通 | 能写 LED 点亮/熄灭/翻转，按键检测（消抖） |
| 外部中断 EXTI | 🔴 精通 | 按键触发中断，理解中断优先级 |
| 串口 USART | 🔴 精通 | printf 重定向到串口，中断收发数据 |
| 定时器 TIM | 🔴 精通 | 定时中断、PWM 控制 LED 亮度/蜂鸣器 |
| ADC | 🟡 会用 | 电位器采样、光敏/温度传感器读数 |
| I2C | 🟡 会用 | OLED 显示文字、温湿度传感器 |
| SPI | 🟡 会用 | SPI Flash 读写 |

### 必做项目（至少完成前 3 个）

| # | 项目 | 覆盖知识点 |
|:--:|------|-----------|
| 1 | **串口控制 LED** | GPIO + 串口接收 + 字符串解析 |
| 2 | **PWM 呼吸灯** | 定时器 + PWM 占空比渐变 |
| 3 | **环境数据采集报警** | ADC + 串口输出 + 阈值判断 |
| 4 | 桌面监测仪 | OLED + I2C 传感器 + 按键切换页面（选做） |

### 软件

| 软件 | 必须 |
|------|:--:|
| STM32CubeIDE | ✅ |
| CH340 驱动 | ✅ |
| 串口助手（SSCOM/MobaXterm） | ✅ |

## 第二阶段：FreeRTOS 实时操作系统

### 网课推荐

| 优先级 | 课程 | 链接 |
|:--:|------|------|
| ⭐⭐⭐ | **正点原子 FreeRTOS 教程** | [B站](https://www.bilibili.com/) |

### 要学到什么地步

| 知识点 | 熟练度 | 标准 |
|--------|:--:|------|
| 任务创建/删除 | 🔴 精通 | 能创建多个任务并发运行 |
| 队列 | 🔴 精通 | 任务间传递数据 |
| 信号量 | 🔴 精通 | 同步多个任务、资源共享 |
| 互斥锁 | 🔴 精通 | 保护临界区 |
| 软件定时器 | 🟡 会用 | 定时回调 |
| 中断+任务通信 | 🔴 精通 | 中断里发信号量，任务里处理 |

### 必做项目

| # | 项目 |
|:--:|------|
| 1 | 多任务 LED：任务1 控制 LED1 闪烁，任务2 控制 LED2，互不干扰 |
| 2 | 串口命令解析：中断收数据 → 队列发给解析任务 → 控制 LED/PWM |

## 第四阶段：Linux 驱动/BSP（核心中的核心）

### 网课推荐

| 优先级 | 课程 | 链接 |
|:--:|------|------|
| ⭐⭐⭐ | **韦东山 Linux 驱动** | [B站搜「韦东山」](https://www.bilibili.com/) |
| ⭐⭐⭐ | **正点原子 I.MX6ULL 驱动** | [B站](https://www.bilibili.com/) |

### 要学到什么地步（分两层）

#### 入门层（能找实习）

| 知识点 | 熟练度 |
|--------|:--:|
| 内核模块编译/加载/卸载 | 🔴 精通 |
| 字符设备驱动（file_operations） | 🔴 精通 |
| 平台设备驱动（platform_driver） | 🔴 精通 |
| 设备树（Device Tree） | 🔴 精通 |
| 中断处理（上半部/下半部） | 🔴 精通 |
| GPIO 子系统 / LED 子系统 | 🟡 会用 |
| I2C 子系统 | 🟡 会用 |
| SPI 子系统 | 🟡 会用 |
| printk 调试 | 🔴 精通 |

#### 进阶层（正式 BSP 岗）

| 知识点 | 熟练度 |
|--------|:--:|
| 并发同步（自旋锁/信号量/互斥体） | 🔴 精通 |
| DMA 与 Cache 一致性 | 🔴 精通 |
| U-Boot 启动流程 | 🔴 精通 |
| 内核内存管理（kmalloc/vmalloc/ioremap） | 🟡 理解 |
| V4L2（摄像头）/ DRM（显示）/ ALSA（音频） | 🟡 了解 |
| ftrace / kgdb 调试 | 🟡 会用 |

### 必做项目（简历项目）

| # | 项目 | 写在简历上 |
|:--:|------|:--:|
| 1 | LED 字符设备驱动 + 按键中断 | ✅ |
| 2 | I2C 温湿度传感器驱动（AHT20/DHT11） | ✅ |
| 3 | SPI OLED 显示驱动 | ✅ |
| 4 | 多设备平台驱动（platform bus 匹配） | ✅ |
| 5 | DMA + UART 高速传输驱动 | ⭐加分项 |
